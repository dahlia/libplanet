GetAddressesMask(HashDigest<SHA256>)

Gets the bitmask of all addresses ever used until the block of the given blockHash.

Declaration
public abstract Address? GetAddressesMask(HashDigest<SHA256> blockHash)
Parameters
Type Name Description
HashDigest<SHA256> blockHash

The Hash of the topmost Block<T> of the Block<T> sequence that updated the Address set to which the bit mask you want to obtain approximates.
Returns
Type Description
Nullable<Address>

A bitmask in the Address type (so this is probably not a valid address). It can be null if the given blockHash refers to a Block<T> that is not in the store.
